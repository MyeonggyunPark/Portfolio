Podo Market Portfolio Project 2

â–¶ Demo video is available on desktop version only.

Podo Market is a fully featured second-hand marketplace built with Python, Django, JavaScript and TailwindCSS. It goes beyond basic CRUD by implementing modern UX features like AJAX interactions and In-Place Editing without relying on heavy frontend frameworks.

  • Interaction System: AJAX-powered Like toggle & Comment system with In-Place Editing (no page reload).
  • Advanced Search: Filter items by title/content with client-side validation.
  • Profile Dashboard: Tabbed UI for [Listings], [Likes], and [Comments] with data deduplication.
  • Scalable Architecture: Uses GenericForeignKey to extend 'Likes' to any content type.
  • Robust Auth: Email verification gate, Social Login (Allauth), and Profile-required middleware.
  • Responsive UI: Mobile-first design using Tailwind CSS.

Deployment is automated using Docker and Railway.

MS Planer Portfolio Project 3

â–¶ Demo video is available on desktop version only.

MS Planer is a full-stack tutor operations platform built with Python, Django, React (Vite), and TailwindCSS. It covers the full tutoring workflow and combines student/course/schedule/exam/todo/invoice management in one integrated system.

  • Core Operations: End-to-end management for students, courses, lessons, mock/official exams, and todos with practical dashboard views.
  • Invoice Workflow: Invoice creation with line items, PDF export, unpaid highlighting, and unsent-notification sync.
  • Auth & Account: JWT cookie auth, email verification/reset flow, and social login integration (Google/Kakao).
  • Internationalization: Frontend and backend i18n support with German/Korean UI, translated messages, and localized email templates.
  • Performance Tuning: Lazy tab loading and optimized querysets (`select_related`/`prefetch_related`) to reduce heavy-page load cost.
  • Production Readiness: Containerized setup with stable deployment configuration and responsive UI across desktop/mobile.
  • UI/UX Focus: The interface is responsive, but the overall UI/UX is optimized primarily for laptop and desktop screens.

Deployment is automated using Docker, Gunicorn, and Railway.
