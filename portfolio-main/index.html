DevJobScraper is a web application built with Python, Flask, and TailwindCSS that scrapes job listings from actual sites like berlinstartupjobs.com, weworkremotely.com, and Stepstone.de allowing users to search jobs based on keywords.

  • Keyword-based search (e.g., Python, JavaScript)
  • Aggregated job counts and site-specific detailed result pages
  • CSV export for downloading job listings
  • Selenium integration for JS-rendered sites like WWR
  • In-memory caching for faster repeated searches
  • Fully responsive UI optimized for mobile and desktop

Deployment is automated using Docker and Railway.
